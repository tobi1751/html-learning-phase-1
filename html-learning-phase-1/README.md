# My-App

Dette projekt er et monorepo, der indeholder flere applikationer og biblioteker.

## Mappestruktur

- **/apps/web**: Indeholder Next.js-applikationen til web.
- **/apps/mobile**: Indeholder React Native-applikationen til mobil.
- **/libs**: Indeholder genbrugelige komponenter, services og utils.
- **/db**: Indeholder database-relateret kode og skemaer.
- **/server**: Indeholder backend-serverlogik (Node.js/Express).
- **/scripts**: Indeholder scripts til build og deploy.
